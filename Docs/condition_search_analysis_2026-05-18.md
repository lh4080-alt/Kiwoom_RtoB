### 키움 조건검색 정확도 평가

| 항목 | 부합 | 부합률 |
|---|---|---|
| A (7봉내 +7%) | 9/12 명확 + 3/12 경계 | 75~100% |
| B (어제 음봉) | 12/12 | **100%** |
| C (오늘 +1%↑) | 12/12 | **100%** |
| E/F (이평 위) | 12/12 | **100%** |

**조건검색이 의도된 "상승지속후 눌림목" 패턴을 정확히 식별 중**. 1차 필터로서 충분한 신뢰성.

### 우리 봇의 역할 위치

```
[1차 필터] 키움 HTS 조건검색
  ↓ "상승지속후 눌림목" 패턴 매칭
[수집] 봇 collection_pool 적재
  ↓
[2차 필터] D-score (7일 캔들 품질 0~10점)
  + 이력 가산 (lookup_history)
  ↓
[3차] 매수 결정 (필터링 모듈 — 향후 구현)
```

1차(HTS 조건검색)가 잘 작동하므로, 우리는 2차/3차에 집중하면 됩니다.
