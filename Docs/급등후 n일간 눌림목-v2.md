| 지표 | 내용 |
| --- | --- |
| A | 주가범위: 0일전 종가가 5000 이상 5000000 이하 |
| B | [일] 0봉전 5봉 평균거래량 100000 이상 999999999 이하 |
| C | 기간내 거래대금: [일] 0봉전 5봉 이내 거래대금(일/주: 백만, 분: 천원) 50000 이상 1회 이상 |
